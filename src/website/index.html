Honest framing: OneCompute does not replace datacenters for frontier training, which needs co-located GPU clusters. It absorbs the large class of substitutable work (evaluation, batch inference, agent runs, synthetic data). Every idle hour harvested is a dollar of cloud spend deferred.

  


  

    

      
A job is signed work. A worker is idle capacity. The orchestrator routes between them.

      
03 · submit → match by capability → fan out → verify → pay. requeue on failure.

    

    

      

        
01
Workers advertise headroom
An opt-in agent on each PC reports spare CPU and GPU, detects idleness, and pulls work over an outbound-only connection.

        
02
The orchestrator routes
It matches each signed job to a suitable idle worker, fans the batch out across the fleet, and requeues anything that drops.

        
03
Results are verified, then paid
Hidden challenge tasks catch bad actors. Employees earn credits for verified useful work, never for claimed performance.

      

      

        
      

    

  


  

    

      

        

          The money shot
          
It never competes with the person at the keyboard.

          
OneCompute rides the learned headroom between what someone uses and what their hardware can do. The moment real work returns it throttles down or kills the job in well under a second, then reroutes it elsewhere.

The worker never sees the job. The job never touches the worker.

      
04 · signed, sandboxed, verified. privacy holds in both directions.

    

    

      

        

          
          
Signed manifests

        

        
Every job ships as an Ed25519-signed manifest. The worker verifies the code hash, data hash, and resource limits before it runs a line. Flip one byte and it refuses.

      

      

        

          
          
Sandboxed, no residue

        

        
Work runs in a per-job container with no network and a read-only payload. Inputs and outputs are wiped when the job ends. The instant-yield kill is a real process-tree kill, not a pause.

      

      

        

          
          
Verified, not claimed

        

        
Hidden challenge tasks with known answers spot-check results. A worker that fakes output is blacklisted and forfeits its points. Credit tracks verified work, never a self-reported number.

      

    

  


  

    

      
The honest number

      
05 · a ceiling worth chasing, always reported beside what we actually measure.

    

    

      

        
1.8 ExaOPS

        
Forty thousand Copilot+ PCs at roughly 45 INT8 TOPS each is about 1.8 ExaOPS of theoretical peak. That is a ceiling at full adoption, not a demo result. A single RTX 5090 laptop alone is about 1,824 TOPS, so a GPU-equipped fleet pushes the real number up fast.
