OneCompute resource kit

Every free, drop-in resource the fleet found, vendored locally and wired up. Hover and click the icons, fire the sounds, spin the 3D model. All offline, no keys.

3D icons — Fluent Emoji 3D (MIT) · GSAP float

Sound — synthesized SFX (Howler) + procedural (Tone.js)

SFX are license-free starters generated with ffmpeg. Swap in Kenney (CC0) or Mixkit (attribution-free) packs for designed sound. Tone.js synthesizes risers/booms with zero files.

3D object — model-viewer (Apache-2.0) + a Khronos GLB

Drop in any GLB

One tag renders a cinematic 3D object with auto lighting and tone-mapping:

<model-viewer src="x.glb" camera-controls auto-rotate>

Pull free CC0 models + HDRIs from Poly Haven, Quaternius, Kenney. Animate camera-orbit with GSAP for a virtual camera move.

Animated backdrop — Vanta.js (MIT, behind this page)

The moving fog behind everything is one Vanta call on #vanta. Also vendored: Granim (gradient), Cobe (globe), tsParticles (particles), lottie-web (Lottie icons).
