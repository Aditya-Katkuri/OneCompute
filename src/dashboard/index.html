+ Connect a device

In the OneCompute repo on any laptop or dev box on the same network, run uv sync once, then run this to join the fleet:

uv run python -m worker --url …
  1. Run the command on the machine you want to contribute.
  2. It appears below as Pending with a short device code.
  3. Click Approve on its tile, and it starts harvesting work.

Approval is required only when the orchestrator runs with --require-approval; otherwise devices join active automatically.
